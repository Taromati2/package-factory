■SSPについて
SSPは「何か」「伺か」互換のデスクトップエージェントプラットホームです．
デスクトップでキャラクタがおはなししてくれたり，
ネットワークを通じていろいろやってくれたりするようなものです．

SSPは，「何か」「伺か」と比較的高い互換性を有し，
現存する多くのキャラクタ（ゴースト）を動作させることができます．

参考→ http://usada.sakura.vg/


■動作環境
SSPは，以下の環境で動作します（多分）．

Windows 98/Me
Windows 2000/XP/Vista/7それ以降のNT系

テスト環境が用意できませんので、Windows 95 / NT4に関しては十分なサポートを
することはできませんが、Internet Explorer 4がインストール済み
（または過去にインストールされた）であればおそらくある程度は動作するはずです。
NT系(NT4,2000,XP)についてはできる限りService Packを当てた環境でお使いください。

■使用条件，配布条件など
SSPはフリーウェアです．
自由に使用してください．

また，再配布など自由に行って構いません．
いわゆる「ゴースト同梱版」の作成については，技術的問題が多いため
推奨はしません（禁止ではありません）．

ただし，ディレクトリ構造の改変ではなく、ファイルの中身自体の改変や，
たとえばデフォルトバルーンのみを切り出して再配布などのように，
一部を切り出しての再配布等は，こちらでのサポートに影響がありますので
控えていただくようお願いいたします．

雑誌などCDROMへの収録の際には一言声をかけていただけるとありがたいです．
